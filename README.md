# App Docs

다국어 지원 개인정보처리방침 및 이용약관 웹사이트

## 기술 스택

React 19 + TypeScript + Vite + i18next + CSS Modules

## 실행

```bash
npm install
npm run dev
```

## 새 앱 추가

1. `src/data/index.ts`에 앱 설정 추가
2. `src/i18n/locales/*.json` 10개 파일에 번역 추가

## 스크립트

```bash
npm run dev        # 개발 서버
npm run build      # 빌드
npm run check:i18n # 번역 검증
```
